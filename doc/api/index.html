FinApp DSL API Documentation

This is the API documentation for the FinApp DSL project.

Namespaces

  • finapp-dsl.core - Core DSL functionality
  • finapp-dsl.loan-topup - Loan top-up domain implementation
  • finapp-dsl.doc-generator - Documentation generation utilities

finapp-dsl.core

The core namespace provides the fundamental DSL functionality including expression types, environment handling, and evaluation.

Functions

  • make-variable [name] - Creates a variable expression
  • make-application [operator operands] - Creates a function application expression
  • make-lambda [parameters body] - Creates a lambda expression
  • make-if [predicate consequent alternative] - Creates a conditional expression
  • create-global-environment [] - Creates a new global environment
  • evaluate [exp env] - Evaluates an expression in the given environment

finapp-dsl.loan-topup

Implementation of the loan top-up domain, including business rules and calculations for loan top-up processes.
