make-variable

(make-variable name)

Creates a variable expression with the given name.

Parameters:

  • name - A string or symbol representing the variable name

Returns:

A map representing a variable expression with :type and :name keys

Example:

(make-variable "customer") 
;; => {:type :variable :name "customer"}

evaluate

(evaluate exp env)

Evaluates an expression in a given environment.

Parameters:

  • exp - The expression to evaluate
  • env - The environment for evaluation

Returns:

The result of evaluating the expression

Example:

(def env (create-global-environment))
(evaluate (make-variable "x") (define-variable! (make-variable "x") 42 env))
;; => 42
